# ゴルフ練習メモアプリ 設計書

---

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | My GOLF |
| プラットフォーム | iOS（Flutter） |
| 作業ディレクトリ | `/Users/maruyamayayoi/test` |
| 主な目的 | ゴルフの練習内容（クラブ・飛距離・球筋・メモ等）をクラブ別に記録・振り返るアプリ |

## 2. 画面一覧

### 起動後

| ID | 画面名 | 備考 |
|----|--------|------|
| S01_01 | スプラッシュ | 起動時に表示 |

### 記録（記録一覧・詳細）

| ID | 画面名 | 備考 |
|----|--------|------|

## 4. モジュール構成

### 記録モジュール

```
Record UI
├── 記録
│   ├── クラブ選択
│   ├── 作成
│   ├── 編集
│   └── 画像と動画の追加/削除
└── 表示
    ├── 一覧
    ├── 詳細
    └── 検索
```

| アプリケーション層（ユースケース） | データ層 |
|----------------------------------|---------|
| GetPracticeClubs（練習するクラブ一覧を取得） | ClubRepository（取得） |
| CreatePracticeMemo（練習記録を登録） | PracticeMemoRepository |
| UpdatePracticeMemo（練習記録を変更） | ├ 保存 |
| AddMedia（画像・動画の追加） | ├ 取得 |
| DeleteMedia（画像・動画の削除） | ├ 削除 |
| DeletePracticeMemo（練習記録を削除） | └ Mediaの取得/保存/削除 |
| GetList（練習記録の一覧を取得） | |
| GetDetail（練習記録の詳細を取得） | |
| ToggleFavorite（お気に入り切り替え） | |

### 検索モジュール

```
Search UI
├── 検索フィルター
└── 検索結果一覧
```

| アプリケーション層 | データ層 |
|------------------|---------|
| Search（検索条件を受け取り練習記録を検索） | PracticeMemoRepository（取得） |

### レポートモジュール

```
Report UI
├── レポート一覧
├── 期間選択
└── クラブ選択
```

| アプリケーション層 | データ層 |
|------------------|---------|
| GetReport（指定条件〈期間・クラブ〉でレポート用データを取得） | PracticeMemoRepository（取得） |

### 設定モジュール

```
Setting UI
├── 設定一覧
├── 練習するクラブ選択（オンボーディングでも使用）
├── カスタムクラブ作成/編集/削除（オンボーディングでも使用）
├── お問い合わせ
├── 利用規約
├── プライバシーポリシー
└── ライセンス
```

| アプリケーション層 | データ層 |
|------------------|---------|
| GetPracticeClubs（クラブ一覧とオン/オフ状態を取得） | ClubsRepository |
| UpdatePracticeClubStatus（クラブ一覧のオン/オフ状態を更新） | ├ 保存 |
| CreateCustomClub（カスタムクラブを作成） | ├ 取得 |
| UpdateCustomClub（カスタムクラブを編集） | └ 削除（カスタムクラブのみ） |
| DeleteCustomClub（カスタムクラブを削除） | |

## 5. 画面遷移

### 起動フロー

```
S01_01 スプラッシュ
    ├── 初回のみ → S01_02 オンボーディング → S02_01 記録一覧（ホーム）
    └── 2回目以降 → S02_01 記録一覧（ホーム）
```

### ホーム（記録一覧）からの遷移

```
S02_01 記録一覧（すべて）
    ├── カードタップ → S02_03 記録詳細
    ├── お気に入りタブ → S02_02 お気に入り一覧
    ├── 検索アイコン → S03_01 検索フィルター → S03_02 検索結果一覧
    ├── 設定アイコン → S06_01 設定一覧
    └── FAB（＋） → S04_01 記録作成_クラブ選択

S02_03 記録詳細
    ├── ⋯ボタン → S02_04 詳細_編集or削除（アクションシート）
    │   ├── 編集 → S04_05 記録編集
    │   └── 削除 → 削除確認ダイアログ → S02_01 記録一覧
    └── 戻る → S02_01 記録一覧
```

### 記録作成フロー

```
S04_01 記録作成_クラブ選択
    └── クラブ選択 → S04_02 記録作成_入力
        ├── 画像・動画追加 → S04_03 画像・動画選択 → S04_04 画像・動画プレビュー → S04_02
        ├── 日付変更 → 日付変更シート → S04_02
        └── 保存 → S02_01 記録一覧

S04_05 記録編集（詳細から編集）
    └── 保存 → S02_03 記録詳細
```

### レポートフロー

```
S05_01 レポート一覧
    ├── 期間選択（30日 / 60日）
    └── クラブ選択ボタンタップ → S05_02 クラブ選択（シート表示）
```

### 設定フロー

```
S06_01 設定一覧
    ├── 練習するクラブ → S06_02 練習するクラブ
    │   └── カスタムクラブ追加 → S06_03 カスタムクラブ
    ├── お問い合わせ → S06_04（外部リンク）
    └── 規約関連 → S06_05（利用規約 / プライバシーポリシー / ライセンス）
```

## 6. データモデル

### PracticeMemo（練習記録）

| カラム名 | 型 | 概要 |
|----------|-----|------|
| practice_memo_id | int | 主キー |
| club_id | int | クラブID（外部キー） |
| practiced_at | DateTime | 練習した日 |
| body | String? | テキストメモ |
| condition | String? | 調子：good / normal / bad |
| distance | int? | 飛距離（yd） |
| shot_shape | String? | 球筋：straight / draw / fade / slice / hook |
| wind | String? | 風：strong / normal / weak / none |
| is_favorite | bool | お気に入りフラグ |
| created_at | DateTime | 作成日時 |

### Club（クラブ）

| カラム名 | 型 | 概要 |
|----------|-----|------|
| id | int | 主キー |
| name | String | クラブ名 |
| is_active | bool | 練習一覧に表示するか |
| is_custom | bool | カスタムクラブかどうか |

### Media（メディア）

| カラム名 | 型 | 概要 |
|----------|-----|------|
| media_id | int | 主キー |
| practice_memo_id | int | 練習記録ID（外部キー） |
| type | String | image / video |
| uri | String | ファイルの保存先パス |
| thumbnail_uri | String? | サムネイルの保存先パス（動画用） |
| created_at | DateTime | 作成日時 |

### 8-5. Typography（タイポグラフィ）

> **ソース：Figma Variables + デザイナー指定テキストスタイル**

フォントファミリー：`Hiragino Sans`（日本語 JP）/ `SF Pro`（英数字 EN、iOS システム自動切替）
スタイルは `AppTypography`（`lib/core/constants/app_typography.dart`）で管理。
**色はスタイルに含めず、使用時に `copyWith` で合成する。**

```dart
// 例
Text('クラブ名', style: AppTypography.jpHeader3.copyWith(color: AppColors.textHigh))
```

#### JP（日本語）— Hiragino Sans

| Figma変数名 | トークン名 | サイズ | ウェイト | 行間 | 用途 |
|------------|-----------|--------|---------|------|------|
| `JP/Header/Header1` | `jpHeader1` | `24px` | W6 (600) | 120% | 画面タイトル・大見出し |
| `JP/Header/Header3` | `jpHeader3` | `18px` | W6 (600) | 120% | カード内クラブ名・AppBarタイトル |
| `JP/Header/Header4` | `jpHeader4` | `16px` | W6 (600) | 120% | 小見出し・ラベル見出し |
| `JP/Header/SubHeader` | `jpSubHeader` | `16px` | W4 (400) | 120% | サブ見出し・ラベル |
| `JP/M/Medium` | `jpMMedium` | `16px` | W5 (500) | 150% | 中サイズ・強調テキスト |
| `JP/M/Regular` | `jpMRegular` | `16px` | W4 (400) | 150% | 中サイズ・標準テキスト |
| `JP/S/Medium` | `jpSMedium` | `14px` | W5 (500) | 150% | 小サイズ・強調テキスト（タブラベル等） |
| `JP/S/Regular` | `jpSRegular` | `14px` | W4 (400) | 150% | 小サイズ・本文・一般テキスト |
